# Cafeteria Online

Projeto desenvolvido utilizando HTML, CSS, JavaScript, Node.js, Express e PostgreSQL.

O sistema permite consultar o cardápio de uma cafeteria através de uma aplicação cliente-servidor, consumindo dados armazenados em um banco de dados PostgreSQL.

---

## Funcionalidades

- Visualizar o cardápio completo.
- Visualizar apenas os cafés.
- Visualizar apenas os doces.
- Exibição de imagens dos produtos.
- Consumo de dados através de uma API REST utilizando `fetch()`.

---

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL
- CORS
